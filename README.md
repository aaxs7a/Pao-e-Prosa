🥖 Pão-e-Prosa
Aplicação mobile desenvolvida em React Native com TypeScript, focada na gestão e experiência de usuário para o contexto de uma padaria. O projeto busca unir uma interface intuitiva com a robustez e segurança que a tipagem estática oferece.

🚀 Tecnologias
React Native: Para o desenvolvimento da interface nativa cross-platform.

TypeScript: Garantindo maior previsibilidade e manutenção do código.

Navegação: (React Navigation) Organização fluida entre as telas da padaria.

🛠️ Como rodar o projeto
Clone o repositório:

Bash
git clone https://github.com/aaxs7a/Pao-e-Prosa.git
Entre na pasta principal:

Bash
cd Pao-e-Prosa/Pao-e-Prosa
Instale as dependências:

Bash
npm install
# ou
yarn install
Inicie a aplicação:

Bash
npx expo start
# ou
npm start
📂 Organização
Atualmente, o código está centralizado na pasta Pao-e-Prosa, estruturado para separar a lógica de negócio dos componentes visuais, facilitando a escalabilidade do sistema conforme novas funcionalidades de vendas ou catálogo forem adicionadas.
